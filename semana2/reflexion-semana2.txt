Es importante que un desarrollador sepa usar la terminal porque permite organizar archivos y carpetas de manera rápida. También ayuda a navegar dentro de un proyecto sin usar el mouse. La terminal facilita crear archivos, mover carpetas y ejecutar comandos importantes. Muchas herramientas de programación funcionan desde la terminal. Además, Git se usa normalmente desde comandos. Aprender a usar la terminal mejora el trabajo y hace más fácil administrar proyectos grandes. También ayuda a trabajar de forma más profesional y ordenada.
